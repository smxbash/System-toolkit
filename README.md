# System-toolkit
Basic script to create directory, backup, and see active process

## Features

* Create a directory inside the home folder
* Show active processes
* Show active users
* Backup a file or folder
* Basic input validation

## How to Run

```bash
chmod +x system-toolkit.sh
./system-toolkit.sh
```

## Concepts Practiced

* `while` loop
* `case` statement
* Conditional tests (`-d`, `-e`)
* Regular expressions
* Command success verification

## Purpose

This project was built to practice and consolidate fundamental Bash scripting concepts.
